# Sample data

This folder contains a **synthetic** demo dataset used by `python -m solarpro`,
the test suite, and the browser front-end.

- **`plant_config.json`** — configuration for a fictional 50 kWc rooftop plant
  (Paris coordinates, 30° tilt, due south).
- **`weather_power.csv`** — two years of hourly weather and power. Weather follows
  realistic seasonal/diurnal patterns; power is generated from a known PVUSA
  relation plus soiling, ageing, an injected ~5-week string-fault anomaly, and
  measurement noise.

> ⚠️ This data is **synthetic, for demonstration and reproducibility only** — it
> contains no real measurements. For real analysis, supply your own Open-Meteo
> location and measured production history.

The dataset is regenerated by `scripts/generate_sample_data.py`.
